# Intro Music Files

Place royalty-free MP3 files here with the following names:

- `upbeat.mp3` — 밝고 경쾌 (bright and upbeat)
- `calm.mp3` — 차분하고 감성적 (calm and emotional)
- `dramatic.mp3` — 드라마틱 (dramatic)
- `exciting.mp3` — 신나는 에너지 (exciting energy)
- `mysterious.mp3` — 신비롭고 신선한 (mysterious and fresh)

## Recommended free sources

- **Pixabay Music**: https://pixabay.com/music/
- **Free Music Archive**: https://freemusicarchive.org/
- **YouTube Audio Library**: https://studio.youtube.com/channel/UC.../music

All files should be:
- Format: MP3
- Duration: at least 10 seconds (only first 6s are used for the intro)
- License: royalty-free / Creative Commons

If a music file is missing, the intro will use silence instead.
